# 변경 이력

## 2026-03-25
- GitHub Pages 검증 실패 원인이던 `_tabs/categories.md`, `_tabs/tags.md` 누락을 복원했습니다.
- 카테고리/태그 상세 페이지에서 상위 `/categories/`, `/tags/` 링크가 깨지던 문제를 해결했습니다.
- 협업 기록 `AGENT_COLLAB_LOG.md`에 배포 오류 원인과 수정 내용을 정리했습니다.
- 로컬 `htmlproofer`는 Windows `libcurl` DLL 누락으로 실행되지 않아, 생성된 `_site` 결과를 수동 확인했습니다.
- 사이드바에서는 `분류` 탭만 남기고, 중복되던 `카테고리`/`태그` 탭은 숨겼습니다. 실제 `/categories/`, `/tags/` 페이지는 유지됩니다.
- 포스트 본문 하단 왼쪽 카드와 하단 고정바 왼쪽 라벨을 `인기글`에서 `관련글`로 변경했습니다.
- 하단 왼쪽 추천 로직을 조회수 기반에서 실제 관련도 기반으로 변경했습니다.
- 관련글 슬롯은 항상 유지하되, 실제 관련글이 있으면 우선 사용하고 없으면 일반 글 fallback으로 채우도록 정리했습니다.
- 댓글 시스템은 최종적으로 다시 `Disqus`로 복원했습니다.
- 포스트 hero 메타뿐 아니라 본문 시작부 메타 줄에도 조회수가 표시되도록 보완했습니다.
- 편집기 배포 버튼이 원격 변경을 자동 동기화한 뒤 푸시하도록 개선했습니다.
- 로컬 전용 `.claude` / 로그 파일이 배포 커밋에 섞이지 않도록 자동 제외했습니다.
