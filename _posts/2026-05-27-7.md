맥북에서 개발 환경을 처음 만들 때 가장 먼저 설치해두면 좋은 것이 Homebrew와 Node.js다.
아래 순서대로 실행하면 Homebrew 설치부터 Node.js 24 확인까지 한 번에 정리할 수 있다.

## 1. Homebrew 설치

터미널을 열고 아래 명령어를 실행한다.
공식 안내는 [Homebrew](https://brew.sh/)에서 다시 확인할 수 있다.

```bash
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```

설치가 끝난 뒤 `brew` 명령어가 바로 안 되면 터미널을 새로 열거나, 설치 마지막에 표시되는 `Next steps` 안내를 실행한다.

## 2. Node.js 24 설치

```bash
brew install node@24
```

`node@24`는 버전이 고정된 공식이다. 설치 후 바로 `node` 명령어가 안 잡히면 아래 명령으로 PATH를 추가한다.

```bash
echo 'export PATH="'$(brew --prefix node@24)'/bin:$PATH"' >> ~/.zshrc
source ~/.zshrc
```

## 3. 설치 확인

```bash
node -v
npm -v
```

예상 결과는 다음과 같다.

```text
v24.16.0
11.13.0
```

## 복습 포인트

- Homebrew는 맥에서 프로그램을 터미널로 설치하게 해주는 패키지 관리자다.
- Node.js는 JavaScript를 브라우저 밖에서도 실행하게 해주는 런타임이다.
- npm은 Node.js와 함께 설치되는 패키지 관리자다.
- `node -v`, `npm -v`는 설치가 제대로 되었는지 확인할 때 자주 쓴다.
